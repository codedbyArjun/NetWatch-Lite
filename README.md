# NetWatch-Lite

NetWatch-Lite is a beginner-friendly Python tool to understand **listening network ports**
and basic networking concepts in a safe and ethical way.

---

## 🔍 Features
- Detects common listening TCP ports
- Displays port, service name, and status
- Clean and readable terminal output
- Localhost-only safe scanning

---

## ▶️ Usage

Run the tool using:
```bash
python main.py
```
